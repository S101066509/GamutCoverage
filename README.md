# 專業色域分析可視化工具 (Color Gamut Analysis Tool)

這是一個基於 **PySide6** 與 **Matplotlib** 開發的專業色域比較與覆蓋率計算工具。它可以幫助使用者在 CIE 1931 色度圖上視覺化不同的色域標準，並計算實測設備相對於標準色域（如 BT.2020, DCI-P3 等）的覆蓋率。

## 核心功能

*   **交互式可視化**：即時繪製 CIE 1931 馬蹄形圖、標準白點 (D65) 以及兩個對比的色域三角形。
*   **自定義輸入**：支援手動輸入 R、G、B 三基色的 (x, y) 座標點。
*   **標準色域預設**：內建多種行業標準色域：
    *   BT.709 / sRGB
    *   DCI-P3
    *   BT.2020
    *   NTSC (1953)
    *   Adobe RGB
*   **覆蓋率計算**：使用向量化重心座標法 (Barycentric Coordinates) 進行高精度的色域重合度運算。
*   **動態數據載入**：直接讀取官方 `CIE_xyz_1931_2deg.csv` 數據，確保繪圖邊界與科學標準一致。

## 專案結構

*   `gamut_app.py`: GUI 主程式，負責介面邏輯與繪圖分發。
*   `cal.py`: 核心運算模組，包含三角形判定與覆蓋率演算法。
*   `config.py`: 配置與樣式表檔案，包含介面外觀設定與標準常數。
*   `CIE_xyz_1931_2deg.csv`: 標準 2 度觀測者色度數據源。
*   `requirements.txt`: 專案依賴庫列表。

## 安裝指南

確保你已安裝 Python 3.9+，然後執行以下指令安裝必要套件：

```bash
pip install -r requirements.txt
```

## 使用說明

1.  執行主程式：
    ```bash
    python gamut_app.py
    ```
2.  在左側面版選擇 **Reference Gamut** 作為比較的基準。
3.  在 **Custom/Test Primaries** 區域輸入實測設備的基色座標。
4.  點擊 **[執行運算並更新圖表]** 按鈕，右側將即時更新地圖與標註，並在下方顯示精確的覆蓋率百分比。

## 技術背景

*   **覆蓋率計算法**：透過在 CIE xy 空間生成 500x500 的高清網格，利用 `Triangle.inside_mask` 判定點位分佈，計算重疊區域與基準區域的比值。
*   **圖形渲染**：使用 Matplotlib 的內嵌 Qt 後端進行高效渲染，搭配自定義 QSS 打造深色專業風格。
